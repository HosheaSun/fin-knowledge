# 存款业务解析

## 存款产品类型

### 活期存款

- 随存随取
- 按季结息
- 利率最低

### 定期存款

| 类型 | 期限 | 特点 |
|------|------|------|
| 整存整取 | 3月/6月/1年/2年/3年/5年 | 到期一次性支取本息 |
| 零存整取 | 1年/3年/5年 | 每月存入，到期取本息 |
| 整存零取 | 1年/3年/5年 | 一次存入，定期支取 |
| 存本取息 | 1年/3年/5年 | 一次存入，定期取息 |

### 通知存款

- 1天通知存款
- 7天通知存款
- 支取前需通知

## 利率定价机制

### 基准利率

央行公布的法定利率

### 存款利率自律机制

- 银行间利率定价自律机制
- 上限约束：一般存款上浮不超过50%
- 考核机制

### 利率计算

```
利息 = 本金 × 年利率 × 存期
```

## 核心系统

- 储蓄系统：活期、定期存款处理
- 利率管理系统：利率参数配置
- 核算系统：账务处理与分录
