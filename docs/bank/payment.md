# 支付结算体系

## 支付系统架构

### 中国现代化支付系统（CNAPS）

| 系统 | 功能 |
|------|------|
| 大额实时支付（HVPS） | 实时全额清算，单笔金额大 |
| 小额批量支付（BEPS） | 批量处理，净额清算 |
| 网上支付跨行清算（IBPS） | 网银跨行转账 |
| 境内外币支付（CFXPS） | 外币跨境清算 |

### 清算所

- 中国银联：银行卡跨行交易
- 城银清：城市商业银行清算
- 农信银：农村金融机构清算

## 支付流程

### 行内转账

```
转出方行内账户 → 人行账户体系 → 转入方行内账户
```

### 跨行转账

```
发起行 → 人民银行业务处理中心 → 接收行
```

## 支付工具

| 工具 | 特点 |
|------|------|
| 银行卡 | 介质支付 |
| 网银支付 | PC端支付 |
| 移动支付 | 手机端支付 |
| 扫码支付 | 商户扫码/用户扫码 |
| 刷脸支付 | 生物识别支付 |

## 风控要点

- 反洗钱监控
- 交易限额控制
- 异常交易识别
- 身份验证多层认证
