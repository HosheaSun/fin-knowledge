# 信用卡业务解析

## 信用卡生命周期

### 发卡

1. 客户申请进件
2. 征信查询与评分
3. 授信额度核定
4. 制卡与邮寄
5. 客户激活

### 用卡

- 透支消费
- 取现
- 分期付款
- 账单分期

### 还款

- 全额还款
- 最低还款额
- 分期还款

## 核心系统

| 系统 | 功能 |
|------|------|
| 发卡系统 | 申请审批、制卡管理 |
| 核心系统 | 账户管理、账务处理 |
| 收单系统 | 商户交易处理 |
| 风控系统 | 交易监控、欺诈检测 |

## 信用卡风险控制

### 贷前风控

- 反欺诈审查
- 信用评分模型
- 收入核实

### 贷后风控

- 交易监控
- 异常交易预警
- 逾期催收管理

## 重要指标

- **不良率**：逾期90天以上占比
- **催收率**：回收率
- **欺诈损失率**：欺诈交易占比
