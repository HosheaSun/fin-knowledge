# 资产负债管理

## 资产负债管理（ALM）

银行综合管理资产和负债的期限、利率、流动性等风险。

## 核心原则

- **安全性**：确保充足资本
- **流动性**：满足提款和贷款需求
- **盈利性**：追求合理回报

## 期限错配管理

### 典型错配

- 负债端：活期存款为主
- 资产端：长期贷款为主

### 缺口分析

```
利率敏感缺口 = 利率敏感资产 - 利率敏感负债
```

### 久期管理

衡量资产/负债对利率变化的敏感性

## 流动性风险管理

### 指标体系

| 指标 | 要求 |
|------|------|
| 流动性比例 | ≥25% |
| 存贷比 | ≤75% |
| 流动性覆盖率 | ≥100% |
| 净稳定资金比例 | ≥100% |

### 应急机制

- 流动性风险应急预案
- 央行再贷款支持
- 同业拆借备付

## 利率风险管理

- **重定价风险**：资产负债期限不匹配
- **收益率曲线风险**：曲线平行移动
- **基准风险**：不同利率变动不一致
