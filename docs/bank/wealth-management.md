# 理财业务解析

## 理财产品分类

### 按收益类型

- **保本固定收益**：本金+约定收益
- **保本浮动收益**：本金保障，收益浮动
- **非保本浮动收益**：不保障本金

### 按投资标的

| 类型 | 投资范围 |
|------|----------|
| 固定收益类 | 债券、存款 |
| 混合类 | 债券+股票 |
| 权益类 | 股票为主 |
| 衍生品类 | 期权期货 |

## 销售流程

1. **产品研发**：设计产品方案
2. **合规报备**：向监管报备产品
3. **销售渠道**：柜面、网银、手机银行
4. **客户适当性**：风险测评匹配
5. **成交确认**：签约扣款

## 资管新规要点

- **打破刚兑**：产品净值化
- **规范资金池**：单独建账、单独核算
- **限制嵌套**：最多一层嵌套
- **期限匹配**：产品期限对接投资资产

## 理财子公司

银行设立理财子公司，实现：
- 风险隔离
- 专业经营
- 监管合规
