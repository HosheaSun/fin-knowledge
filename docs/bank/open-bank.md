# 开放银行架构

## 什么是开放银行

银行通过API向第三方合作伙伴开放金融服务，构建银行生态圈。

## 监管政策

- 央行《金融科技发展规划》
- 银保监会相关指引
- 数据安全法合规要求

## API类型

| 类型 | 说明 |
|------|------|
| 账户服务 | 余额查询、交易明细 |
| 支付服务 | 快捷支付、代扣代付 |
| 贷款服务 | 预审批、在线贷款 |
| 信用卡服务 | 申请、还款、积分 |

## 开放银行架构

### 开放平台层

- API网关

### 服务编排层

### 安全层
