# 投行业务解析

## 投资银行主要业务

### 股权融资

| 业务 | 说明 |
|------|------|
| IPO | 首次公开发行股票 |
| 增发 | 向特定或不特定对象增发 |
| 配股 | 向原股东配发新股 |
| 可转债 | 发行可转换债券 |

### 债券融资

- 企业债
- 公司债
- 金融债
- ABS资产证券化

### 并购重组

- 财务顾问
- 估值服务
- 交易结构设计

## IPO流程

### 准备阶段

1. 尽职调查
2. 股份制改造
3. 辅导期（1年以上）
4. 材料申报

### 审核阶段

1. 证监会受理
2. 反馈意见
3. 初审会
4. 发审会

### 发行上市

1. 询价路演
2. 定价
3. 摇号配号
4. 上市交易

## 核心系统

- **承销系统**：项目跟踪管理
- **估值系统**：资产评估
- **合规系统**：材料审核
