# 股票交易流程

## 交易流程概览

```
投资者下单 → 券商接收 → 交易所撮合 → 成交回报 → 清算交收
```

## 委托下单

### 委托方式

| 方式 | 说明 |
|------|------|
| 限价委托 | 指定价格成交 |
| 市价委托 | 按最优价格成交 |
| 止损委托 | 触发价成交 |

### 委托要素

- 证券代码
- 买卖方向
- 委托数量
- 委托价格
- 股东账户

## 交易所撮合

### 竞价原则

- **价格优先**：买入时高价优先，卖出时低价优先
- **时间优先**：相同价格，先提交的委托优先

### 撮合机制

- 集合竞价：9:15-9:25（开盘）、15:00（收盘）
- 连续竞价：交易时间段

## 清算交收

### T日流程

1. **T日交易**：正常买卖股票
2. **T日结算**：收市后清算
3. **T+1交收**：资金和股份过户

### 结算内容

- 印花税：卖出收取0.1%
- 过户费：沪市收取0.01%
- 券商佣金：不超过成交额0.3%

## 核心系统

- **经纪柜台系统**：接收客户委托
- **交易所交易系统**：撮合引擎
- **登记结算系统**：股份资金清算
