# 证券清算结算体系

## 清算 vs 结算

| 环节 | 说明 |
|------|------|
| 清算 | 计算应收应付资金和证券 |
| 交收 | 资金和证券的实际过户 |

## 中国证券清算体系

### 登记结算机构

| 机构 | 职责 |
|------|------|
| 中证登 | 沪市、深市登记结算 |
| 上交所 | 竞价交易清算 |
| 深交所 | 竞价交易清算 |
| 银证转账 | 资金往来清算 |

### 两级清算

#### 一级清算（交易所→券商）

- 交易所与结算参与人之间
- T日收市后进行

#### 二级清算（券商→投资者）

- 券商与客户之间
- T+1日完成

## 结算流程

### T日

1. **交易完成后**：接收成交数据
2. **日终清算**：计算资金和股份
3. **清算数据**：发送各参与方

### T+1日

1. **资金交收**：通过银行备付金账户
2. **股份交收**：通过中证登账户体系
3. **交收确认**：发送交收单据

## 结算差错处理

- 发现差错后及时报告
- 与交易方协商处理
- 重大差错上报监管

## 系统架构

```
交易所系统 → 结算系统 → 券商柜台 → 银行系统
     ↓           ↓           ↓         ↓
  成交数据    清算数据    客户账户   资金账户
```
