# 融资融券业务

## 业务定义

### 融资业务

投资者向券商借钱买股票，预期股价上涨后卖出获利

```
融资买入 → 持有股票 → 卖出还款
```

### 融券业务

投资者向券商借股票卖出，预期下跌后买回还券

```
融券卖出 → 持有现金 → 买券还券
```

## 准入管理

### 融资融券准入条件

| 条件 | 要求 |
|------|------|
| 交易经验 | ≥6个月 |
| 资产门槛 | 50万以上 |
| 风险测评 | 积极型以上 |
| 信用评估 | 无重大违约 |

### 标的证券

- 在交易所上市的A股
- 交易所公布的可充抵保证金证券名单
- 券商风险评估后的自有标的范围

## 保证金管理

### 保证金比例

```
融资保证金比例 = 保证金 / (融资买入金额 × 融资保证金比例)
融券保证金比例 = 保证金 / (融券卖出金额 × 融券保证金比例)
```

### 维持担保比例

| 比例区间 | 处理方式 |
|----------|----------|
| ≥300% | 提取超过300%的部分 |
| 150%~300% | 正常交易 |
| 130%~150% | 追加保证金 |
| 低于130% | 强制平仓 |

## 风控要点

- 集中度控制（单只股票持仓比例）
- 杠杆率控制
- 强制平仓流程
- 追保通知机制
