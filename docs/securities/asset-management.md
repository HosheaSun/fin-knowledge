# 资产管理业务

## 券商资管业务类型

### 集合资产管理

- 多个投资者资金集合
- 设立产品管理
- 门槛：单一产品100万起

### 定向资产管理

- 单个客户委托
- 一对一服务
- 通道类业务为主

### 专项资产管理

- 资产证券化
- 私募REITs
- 资产支持票据

## 产品设计流程

1. **市场需求调研**
2. **产品方案设计**
3. **合规审查**
4. **产品报备**
5. **销售发行**

## 投资运作

### 投资范围

- 股票
- 债券
- 基金
- 衍生品
- 现金管理工具

### 风险控制

- 止损线设置
- 持仓限制
- 流动性管理

## 运营管理

| 环节 | 内容 |
|------|------|
| 估值核算 | 单位净值计算 |
| 信息披露 | 定期报告 |
| 份额管理 | 申赎处理 |
| 佣金结算 | 管理费提取 |

- 证监会资管新规
- 券商资管八条底线
