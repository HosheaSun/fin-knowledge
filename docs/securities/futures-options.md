# 期货与期权业务

## 期货基础

### 期货合约要素

- 标的资产
- 合约月份
- 交易单位
- 报价单位
- 最小变动价位

### 期货 vs 现货

| 特征 | 期货 | 现货 |
|------|------|------|
| 交易对象 | 标准化合约 | 实物/证券 |
| 成交方式 | 集中撮合 | 随时成交 |
| 交割方式 | 到期交割 | 即时交割 |
| 资金要求 | 保证金 | 全额资金 |

## 期权基础

### 期权类型

- **认购期权（Call）**：买入权利
- **认沽期权（Put）**：卖出权利

### 期权 vs 期货

| 特征 | 期权 | 期货 |
|------|------|------|
| 买方权利 | 有权无义务 | 双方都有义务 |
| 卖方义务 | 被动履约 | 主动履约 |
| 收益特征 | 非线性 | 线性 |
| 最大亏损 | 权利金 | 无限（期货） |

## 保证金管理

### 期货保证金

```
保证金 = 合约价值 × 保证金比例
```

### 维持保证金

- 账户权益必须高于维持保证金
- 低于时需追缴保证金
- 严重时强制平仓

## 风控要点

- 保证金不足实时监控
- 大户报告制度
- 持仓限额制度
- 强行平仓机制
