# 量化投资架构

## 量化交易流程

```
策略研发 → 策略回测 → 仿真交易 → 实盘交易 → 绩效分析
```

## 量化策略类型

### 市场中性

- Alpha对冲
- 统计套利
- 期现套利

### 趋势跟踪

- 均线策略
- 动量策略
- 海龟策略

### 高频交易

- 、做市商策略
- 统计套利
- 微观结构策略

## 系统架构

### 行情模块

- 实时行情接收
- 数据清洗
- 因子计算

### 交易模块

- 策略信号
- 订单生成
- 订单执行
- 仓位管理

### 风控模块

- 事前预警
- 事中监控
- 事后分析

## 核心技术

| 组件 | 技术选型 |
|------|----------|
| 策略开发 | Python, Jupyter |
| 回测框架 | Backtrader,聚宽 |
| 仿真交易 | XTP, MiniTick |
| 实盘交易 | FPGA, 低延迟网络 |
| 风控 | 实时计算引擎 |

## 数据因子体系

- **基本面因子**：估值、成长、盈利
- **技术因子**：均线、MACD、RSI
- **另类因子**：舆情、卫星数据
