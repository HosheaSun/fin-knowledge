# ETF基金业务解析

## ETF概述

ETF（Exchange Traded Fund）是交易型开放式指数基金，结合了封闭式基金和开放式基金的优点，可以在交易所上市交易。

## 核心机制

### 申购赎回机制

- 投资者使用一篮子股票申购ETF份额
- 赎回得到的是一篮子股票而非现金
- 套利机制使ETF价格接近净值

### 一级市场

- 机构投资者参与申购赎回
- 最小申购赎回单位通常为50万份
- 实物申赎减少现金拖累

### 二级市场

- 在交易所实时交易
- 交易价格接近基金净值
- 支持当日买卖

## 业务流程

1. 投顾输出指数权重
2. 基金持仓根据权重构建
3. 每日公布参考净值（IOPV）
4. 做市商提供流动性
5. 套利者平滑折溢价

## ETF类型

| 类型 | 特点 |
|------|------|
| 宽基ETF | 跟踪宽基指数 |
| 行业ETF | 专注特定行业 |
| 债券ETF | 投资债券市场 |
| 黄金ETF | 跟踪黄金价格 |
