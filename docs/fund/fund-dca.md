# 基金定投业务解析

## 什么是基金定投

定投是指定期定额投资开放式基金，类似银行的零存整取，通过长期平均成本分散风险。

## 核心优势

- **淡化择时**：避免追涨杀跌
- **平均成本**：高点买入少，低点买入多
- **纪律投资**：强制储蓄，积少成多
- **门槛低**：几百元即可参与

## 定投协议要素

| 要素 | 说明 |
|------|------|
| 扣款金额 | 每期固定金额 |
| 扣款日 | 每月约定日期 |
| 基金品种 | 可选择多只基金 |
| 终止条件 | 约定期数完成或主动终止 |

## 扣款流程

1. **T-1日**：系统检查定投协议状态
2. **T日**：执行扣款
3. **T+1日**：确认申购份额
4. **T+2日**：份额可查询

## 常见策略

### 普通定投
固定金额、固定日期

### 智能定投
- 指数估值定投（低估值多买）
- 均线定投（偏离均线时调整金额）

### 目标定投
设定收益目标，达到后自动赎回
