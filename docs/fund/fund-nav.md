# 基金净值计算与披露

## 基金净值基本概念

### 单位净值

```
单位净值 = (基金资产 - 基金负债) / 基金份额
```

### 累计净值

```
累计净值 = 单位净值 + 历史分红
```

## 计算流程

1. **收盘后获取行情数据**
   - 股票收盘价
   - 债券估值价
   - 汇率收盘价

2. **计算基金资产**
   - 股票：市值 = 股数 × 收盘价
   - 债券：按估值价计算
   - 银行存款：按账面价值

3. **计提费用**
   - 管理费
   - 托管费
   - 销售服务费

4. **计算净值**
   - 生成各类份额净值
   - 发布净值公告

## 净值披露规范

| 类型 | 披露频率 | 披露时间 |
|------|----------|----------|
| 日净值 | 每日 | T日收盘后 |
| 万份收益 | 每日（货币基金） | T日 |
| 季报 | 每季度 | 季度结束后15个工作日 |
| 半年报 | 每半年 | 上半年结束后60日 |
| 年报 | 每年 | 年度结束后90日 |

## 估值原则

- **上市股票**：收盘价
- **流通受限股**：限售期价格
- **债券**：第三方估值
- **未上市股票**：成本价或估值模型
