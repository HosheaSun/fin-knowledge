# 基金分红与拆分

## 基金分红

### 分红条件

- 分红后净值需高于面值
- 需弥补往年亏损
- 持有足够未分配利润

### 分红方式

#### 现金分红

- 红利直接发放到银行卡
- 不收取赎回费
- 适合稳健投资者

#### 红利再投资

- 分红自动申购为基金份额
- 不收取申购费
- 适合长期投资者

### 分红流程

1. 基金管理人公告分红方案
2. 权益登记日确认持有人
3. 除息日净值相应下降
4. 红利发放日支付

## 基金拆分（折算）

### 目的

- 降低净值，便宜投资者购买
- 解决"净值恐高"问题
- 优化持有人结构

### 方式

| 类型 | 说明 |
|------|------|
| 份额拆分 | 按比例降低每份净值，增加份额 |
| 大比例分红 | 大比例分红后自然降低净值 |

### 折算日流程

- 净值归一
- 份额按比例增加
- 总资产不变
