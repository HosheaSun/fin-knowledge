# 基金转换与赎回

## 基金赎回

### 赎回流程

1. 投资者提交赎回申请
2. 系统确认持有份额
3. 计算赎回金额
4. 资金划转到账

### 赎回费用

```
赎回费用 = 赎回份额 × 净值 × 赎回费率
```

- 持有少于7天：不低于1.5%
- 持有7天及以上：按合同约定费率
- 同一基金公司旗下转换：不收取赎回费

### 到账时间

| 类型 | 到账时间 |
|------|----------|
| 货币基金 | T+1 |
| 债券基金 | T+3~T+4 |
| 股票基金 | T+5~T+7 |

## 基金转换

### 转换优势

- 节省时间（T+1确认）
- 节省费用（只补差费率）
- 资金效率高

### 转换规则

```
转换费 = 赎回费 + 申购补差费
```

- 申购补差 = |转入基金申购费 - 转出基金申购费|
- 持有期合并计算

### 转换限制

- 同一家基金公司
- 同一种基金类型
- 正常开放状态
