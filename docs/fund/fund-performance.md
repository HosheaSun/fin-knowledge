# 基金业绩评价体系

## 收益指标

### 区间收益

- **区间收益率**：期末净值/期初净值 - 1
- **年化收益率**：考虑复利效应的年收益

### 超额收益

```
超额收益 = 基金收益率 - 基准收益率
```

## 风险指标

| 指标 | 公式/含义 |
|------|------------|
| 标准差 | 收益率波动程度 |
| 最大回撤 | 历史最大亏损 |
| 夏普比率 | 风险调整后收益 |
| 卡玛比率 | 收益/最大回撤 |

## 经典业绩归因模型

### Brinson模型

将基金超额收益归因于：

- **资产配置收益**：大类资产配置决策
- **行业选择收益**：行业内选股贡献
- **个股选择收益**：个股选择贡献
- **交互收益**：配置与选股的交互

### 公式

```
超额收益 = 配置收益 + 选择收益 + 交互收益
```
