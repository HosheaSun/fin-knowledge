# 基金风控体系

## 风控框架

### 三道防线

1. **第一道**：业务部门自查
2. **第二道**：风险管理部监控
3. **第三道**：内部审计监督

## 合规风控要点

### 投资限制

- 单只股票不超过基金资产10%
- 同一发行人债券不超过10%
- 流动性持仓不低于5%

### 双十规定

- 单只股票市值不超过10%
- 单只股票占流通股不超过10%

### 反洗钱

- 客户身份识别（KYC）
- 可疑交易监控
- 大额交易报告

## 风险监控指标

| 类型 | 指标 |
|------|------|
| 市场风险 | VaR、跟踪误差 |
| 信用风险 | 持仓债券评级分布 |
| 流动性风险 | 持仓变现能力 |
| 操作风险 | 交易差错率 |

## 估值风控

- 估值方法合规性
- 净值计算准确性
- 持仓流动性监测
